# MongoDB Connection Lab

## Overview
This project demonstrates how to connect a Node.js and Express application to a MongoDB Atlas database using environment variables for security.

## Technologies Used
- Node.js
- Express
- MongoDB Atlas
- MongoDB Driver
- dotenv

## Features
- Connects to a MongoDB Atlas cluster
- Uses environment variables to secure credentials
- Runs an Express server on port 3001
- Returns a successful database connection message at the root route
